Crime Data Analysis Project Explanation

I conducted an analysis on a dataset related to crimes against women in India from 2001 to 2021, covering categories such as rape, kidnapping & abduction, dowry deaths, assaults, domestic violence, and more. I used Python with libraries like Pandas for data manipulation, NumPy for numerical operations, Matplotlib, and Seaborn for data visualization.

Data Preparation

I loaded the dataset using Pandas and conducted an initial review using methods like df.head() to see the first few rows and understand the data structure. I noticed there was an unnecessary column, ‘Unnamed: 0’, which I removed. I also verified that there were no missing values or duplicated entries by using df.isna().sum() and df.duplicated().sum().

I used df.info() to check the data types and confirmed that the dataset was ready for analysis.

Data Analysis

Using df.describe(), I identified the mean, median, and standard deviation for different crime categories. This helped to understand the general trend and distribution of data across years.

Handling Unique Values

There were some inconsistencies in the 'State' column, where states were repeated with different capitalizations (like 'MADHYA PRADESH' and 'Madhya Pradesh'). I grouped them correctly for accurate analysis.

State-Level Analysis

  • I analyzed crime patterns at the state level. For example, I found that Madhya Pradesh and Rajasthan had the highest recorded incidents of rape, with counts of 43,552 and 31,850, respectively.
  • I visualized the number of rape cases over the years using a bar chart, helping to highlight trends over time.

Kidnapping & Abduction (K&A) and Dowry Deaths (DD) Analysis

I aggregated the data for kidnapping & abduction and dowry deaths. Uttar Pradesh topped both categories, with a total of 123,058 incidents. I created a new column, ‘Total’, by adding the values from ‘K&A’ and ‘DD’ to see the combined impact of these crimes.

Assault Analysis

For the ‘Assault on Women’ category, I plotted a pie chart to show the distribution by state. The top states accounted for a significant portion of the incidents.

Other Crimes Against Women

I also analyzed other crimes like ‘Assault on Men’ (AoM) and grouped the data to show the states with the highest incidents. Andhra Pradesh and Uttar Pradesh had the most reported cases.

Visualizations and Insights

I used bar charts, pie charts, and other visualizations to make the data easily understandable and to identify trends and outliers. These visual insights were crucial in interpreting the data.

For instance, the pie chart on assaults clearly showed that a few states accounted for a large percentage of total incidents.

Challenges and Solutions

One challenge was dealing with inconsistent state names due to variations in capitalization. I addressed this by grouping and aggregating the data accurately.

Fortunately, there were no missing values, which simplified the analysis.

The analysis revealed that Madhya Pradesh and Uttar Pradesh are particularly high in crimes like rape and kidnapping, indicating a need for focused interventions in these regions.

These insights can help policymakers and law enforcement agencies allocate resources more effectively and target areas with high crime rates.

Technical Skills Showcased

This project highlighted my ability to clean, manipulate, and analyze complex datasets using Python. I also demonstrated proficiency in creating visualizations that convey clear and impactful insights.

These skills are crucial for a Data Analyst, as they enable me to turn raw data into actionable insights that can drive decision-making.
